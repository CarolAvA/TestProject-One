# 《反幸存者：造物主试炼》玩法策划案

> 版本：V0.1  
> 类型：反向幸存者 / 轻策略 / 肉鸽对抗 / 自动战斗  
> 核心原型参考：《吸血鬼幸存者》  
> 核心创新：玩家不控制英雄，而是扮演造物主，通过布置地图、召唤怪物、制造克制组合，击败由 AI 控制并不断成长的英雄。

---

## 1. 项目概述

### 1.1 一句话概念

传统幸存者游戏中，玩家控制英雄在怪潮中活下去。  
本作反过来：玩家控制怪物、障碍、陷阱和战场规则，目标是让 AI 英雄活不下去。

---

### 1.2 游戏定位

| 项目 | 内容 |
|---|---|
| 游戏类型 | 反向幸存者 + 自动战斗 + 策略召唤 + 肉鸽克制 |
| 玩家身份 | 造物主 / 魔王 / 试炼设计者 |
| 玩家操作对象 | 地图、障碍、陷阱、怪物、精英、Boss、环境规则 |
| AI 操作对象 | 英雄角色 |
| 核心目标 | 在限定时间内击败 AI 英雄 |
| 失败条件 | 英雄存活到时间结束，或英雄完成关卡目标 |
| 单局时长 | 8 ~ 15 分钟 |
| 平台方向 | 移动端优先，也可适配 PC |
| 主要爽点 | 观察英雄流派，使用克制兵种和地形设计，把英雄逼入绝境 |

---

## 2. 核心玩法循环

```text
赛前布置
↓
点击开始试炼
↓
AI 英雄自动战斗
↓
玩家使用士兵抽屉召唤怪物
↓
英雄击杀怪物获得经验并自动升级
↓
英雄选择肉鸽卡牌形成流派
↓
玩家观察流派并召唤克制兵种
↓
通过怪物、陷阱、地形、Boss 压制英雄
↓
英雄死亡：玩家胜利
英雄存活：玩家失败
```

---

## 3. 核心创新点

### 3.1 玩家不再控制英雄

玩家不是移动、攻击、闪避的执行者。  
这些全部交给 AI 英雄处理。

玩家负责的是全局策略：

- 开局布置战场。
- 调整怪物组合。
- 控制刷怪节奏。
- 观察英雄流派。
- 选择克制兵种。
- 设计包围、消耗、爆发和 Boss 决胜时机。

---

### 3.2 英雄是“敌方幸存者玩家”

AI 英雄相当于一个会自动玩《吸血鬼幸存者》的敌方玩家。

英雄会：

- 自动移动。
- 自动攻击。
- 自动释放技能。
- 自动拾取经验。
- 自动升级。
- 自动选择肉鸽卡牌。
- 自动躲避危险。
- 自动寻找安全区域。
- 自动攻击高威胁目标。

玩家要做的是反过来研究这个英雄，并击败它。

---

### 3.3 玩家是“关卡导演”

玩家不是简单刷怪，而是像设计一场死亡试炼：

- 哪里放障碍？
- 哪里放召唤门？
- 什么时候放低级怪试探？
- 什么时候停手避免给英雄送经验？
- 英雄走吸血流时怎么禁疗？
- 英雄走护盾流时怎么破盾？
- 英雄走范围清怪流时怎么用高血怪和远程怪反制？

---

## 4. 单局流程设计

## 4.1 阶段一：赛前布置阶段

游戏开始后，玩家先进入赛前布置阶段。  
此阶段英雄还没有开始行动，玩家可以改造战场。

### 可布置内容

| 类型 | 作用 |
|---|---|
| 障碍物 | 改变英雄移动路线，制造绕路和卡位 |
| 墙体 | 阻挡移动，但不能完全封死英雄 |
| 尖刺陷阱 | 英雄经过时受到一次伤害 |
| 毒沼 | 持续伤害并降低移动速度 |
| 减速阵 | 降低英雄移速，适合配合远程怪 |
| 召唤门 | 后续怪物可从该点出现 |
| 怪物巢穴 | 定期自动刷出低级怪 |
| 资源祭坛 | 提升玩家召唤能量恢复速度 |
| 黑雾区 | 降低英雄索敌范围 |
| 精英巢穴 | 中后期可召唤精英怪 |

---

### 布置限制

| 限制 | 说明 |
|---|---|
| 布置点数限制 | 开局不能无限摆放 |
| 英雄出生点保护区 | 英雄出生点附近禁止放置陷阱和怪物门 |
| 不能完全封死路线 | 必须保证英雄拥有可移动路径 |
| 单位占格 | 所有障碍、陷阱、建筑都占据格子 |
| 高级物件消耗更高 | 毒沼、召唤门、祭坛等消耗更多点数 |
| 地图基础结构固定 | 玩家是在基础地图上二次改造 |

---

### 赛前布置目标

玩家在这个阶段不是直接击杀英雄，而是设计后续战斗环境。

常见思路：

1. 用墙体制造绕路。
2. 用毒沼和减速区限制英雄走位。
3. 用召唤门形成多方向包围。
4. 用障碍保护远程怪输出。
5. 用陷阱区配合控制怪打爆发。
6. 用资源祭坛提高中后期能量恢复。
7. 用黑雾区降低英雄清怪效率。

---

## 4.2 阶段二：正式战斗阶段

玩家点击“开始试炼”后，AI 英雄开始自动行动。

此阶段玩家主要通过下方“士兵抽屉”召唤怪物攻击英雄。

---

### 玩家可执行操作

| 操作 | 说明 |
|---|---|
| 召唤普通怪物 | 使用少量能量制造压力 |
| 召唤远程怪物 | 在远处持续消耗英雄 |
| 召唤坦克怪物 | 卡位、挡路、吸收伤害 |
| 召唤功能怪物 | 禁疗、破盾、减速、控制 |
| 召唤精英怪 | 制造关键压制窗口 |
| 召唤 Boss | 作为终局杀招 |
| 激活陷阱 | 部分陷阱可手动触发 |
| 强化召唤点 | 提升某区域怪物刷出强度 |
| 使用造物主权能 | 短时间改变战局 |

---

## 4.3 阶段三：后期决胜阶段

当英雄成长到一定程度后，玩家进入后期决胜阶段。

此时英雄可能已经拥有完整流派，普通怪物很难造成威胁。  
玩家需要使用更高强度的组合手段。

### 后期策略

| 策略 | 说明 |
|---|---|
| 精英组合 | 多个精英配合普通怪形成压制 |
| 地形围杀 | 利用障碍、毒沼、减速区制造死角 |
| Boss 决战 | 在英雄状态较差时召唤 Boss |
| 资源爆发 | 长时间攒能量后进行一波高强度召唤 |
| 功能克制 | 禁疗、破盾、沉默、减速等组合 |
| 分批召唤 | 避免一次性送经验，持续消耗英雄 |

---

## 5. AI 英雄系统

## 5.1 英雄基础行为

AI 英雄应模拟普通幸存者玩家的操作逻辑。

| 行为 | 说明 |
|---|---|
| 自动移动 | 根据怪物密度、危险区域、经验球位置选择移动方向 |
| 自动攻击 | 自动释放武器和技能 |
| 自动拾取经验 | 优先拾取安全范围内的经验 |
| 自动升级 | 升级后从卡牌池中选择强化方向 |
| 自动躲避危险 | 避开毒沼、火焰、陷阱、Boss 技能 |
| 自动攻击高威胁目标 | 优先攻击远程怪、禁疗怪、破盾怪、召唤门 |
| 自动寻找补给 | 血量低时优先靠近回血道具 |
| 自动突破包围 | 被围住时优先向薄弱方向移动 |

---

## 5.2 英雄基础属性

| 属性 | 作用 |
|---|---|
| 生命值 | 归零后玩家胜利 |
| 攻击力 | 决定基础伤害 |
| 技能伤害 | 决定武器和技能伤害 |
| 攻击频率 | 影响输出频率 |
| 冷却缩减 | 减少技能冷却 |
| 移动速度 | 影响走位和逃生能力 |
| 拾取范围 | 影响经验和补给获取 |
| 护甲 | 减少物理伤害 |
| 魔抗 | 减少法术伤害 |
| 闪避 | 概率规避攻击 |
| 暴击率 | 提高爆发能力 |
| 吸血 | 攻击或击杀回血 |
| 护盾值 | 额外承伤层 |
| 控制抗性 | 减少减速、眩晕等效果 |

---

## 5.3 英雄升级与肉鸽卡牌

英雄击杀怪物后获得经验。  
经验满后升级，并自动选择一张肉鸽卡牌。

### 卡牌类型

| 类型 | 说明 |
|---|---|
| 武器卡 | 解锁或强化主要攻击方式 |
| 属性卡 | 提升攻击、生命、移速、护甲等 |
| 生存卡 | 提供回血、吸血、护盾、减伤 |
| 范围卡 | 提升攻击范围和清怪能力 |
| 特殊卡 | 改变技能机制 |
| 流派核心卡 | 决定英雄主要成长方向 |

---

## 5.4 英雄流派设计

### 流派一：火焰范围流

| 项目 | 内容 |
|---|---|
| 核心表现 | 英雄周围产生持续火圈，快速清理近身小怪 |
| 优势 | 清杂兵强，安全区域大 |
| 弱点 | 怕高血量怪、远程怪、火抗怪、分散压力 |
| 玩家反制 | 减少低血小怪，召唤坦克、远程、火抗精英 |

---

### 流派二：飞剑穿透流

| 项目 | 内容 |
|---|---|
| 核心表现 | 飞剑直线穿透，多方向攻击 |
| 优势 | 对直线怪群杀伤高 |
| 弱点 | 怕快速近身、分散包围、背后突袭 |
| 玩家反制 | 召唤高速小怪、刺客怪，从多个方向分散进攻 |

---

### 流派三：吸血续航流

| 项目 | 内容 |
|---|---|
| 核心表现 | 击杀怪物或攻击怪物回血 |
| 优势 | 持久战强，怪越多越能回血 |
| 弱点 | 怕禁疗、爆发、控制、少量高质量怪 |
| 玩家反制 | 少放低血小怪，召唤禁疗怪和爆炸怪 |

---

### 流派四：护盾防御流

| 项目 | 内容 |
|---|---|
| 核心表现 | 定期获得护盾，容错率高 |
| 优势 | 不怕单次爆发 |
| 弱点 | 怕破盾、持续伤害、毒伤、拖冷却 |
| 玩家反制 | 召唤破盾怪、毒虫，等护盾消失后爆发 |

---

### 流派五：召唤物辅助流

| 项目 | 内容 |
|---|---|
| 核心表现 | 英雄召唤宠物或分身协助作战 |
| 优势 | 持续输出高，能分担怪物压力 |
| 弱点 | 怕范围爆炸、召唤物猎手、持续毒圈 |
| 玩家反制 | 爆炸怪、范围毒怪、专打召唤物的怪物 |

---

### 流派六：高机动风筝流

| 项目 | 内容 |
|---|---|
| 核心表现 | 移速高、闪避高、持续拉扯 |
| 优势 | 很难被包围 |
| 弱点 | 怕减速、控制、障碍封路、远程封锁 |
| 玩家反制 | 减速区、控制怪、远程怪、墙体限制路线 |

## 6. 玩家召唤系统

## 6.1 士兵抽屉 UI

屏幕下方显示横向抽屉，玩家通过它召唤怪物。

```text
┌──────────────────────────────────────┐
│ 当前召唤能量：120 / 200               │
├──────────────────────────────────────┤
│ 分类：小怪 / 远程 / 坦克 / 刺客 / 法术 / 精英 │
├──────────────────────────────────────┤
│ [骷髅兵] [毒虫] [弓箭手] [石甲怪] [爆裂虫] │
│  10能量   15能量   25能量   40能量   45能量 │
└──────────────────────────────────────┘
```

---

## 6.2 召唤操作

| 操作 | 效果 |
|---|---|
| 点击兵种头像 | 选中兵种 |
| 点击地图位置 | 在指定位置召唤 |
| 拖拽兵种到地图 | 快速召唤 |
| 连点兵种头像 | 快速连续召唤 |
| 长按兵种头像 | 打开兵种详情 |
| 点击分类页签 | 切换兵种列表 |
| 点击精英按钮 | 召唤精英怪 |
| 点击 Boss 按钮 | 召唤 Boss |

---

## 6.3 召唤限制

| 限制 | 说明 |
|---|---|
| 能量限制 | 每个兵种有召唤消耗 |
| 冷却限制 | 强力兵种不能连续召唤 |
| 范围限制 | 不能直接贴脸召唤在英雄身上 |
| 数量限制 | 场上同类怪物有数量上限 |
| 解锁限制 | 高级兵种需要时间或条件解锁 |
| 精英限制 | 精英怪一局可召唤次数有限 |
| Boss 限制 | Boss 通常一局只能召唤一次 |

## 7. 召唤能量系统

## 7.1 基础规则

| 项目 | 说明 |
|---|---|
| 初始能量 | 开局给一定能量 |
| 自动恢复 | 每秒恢复少量能量 |
| 上限 | 能量有最大值 |
| 战斗成长 | 时间越久，恢复速度越快 |
| 祭坛加成 | 赛前布置资源祭坛可提升恢复 |
| 连续召唤惩罚 | 短时间大量召唤会提高消耗 |
| 威胁补偿 | 英雄越强，玩家能量恢复越快 |

---

## 7.2 能量设计目的

召唤能量不是单纯的蓝条，而是限制玩家策略的重要资源。

它的作用：

1. 防止无脑刷怪。
2. 让玩家在小怪铺场和精英爆发之间取舍。
3. 让召唤时机变得重要。
4. 避免玩家无限堆怪卡死英雄。
5. 让玩家因为错误召唤付出代价。
6. 形成“攒能量 → 爆发召唤 → 再攒能量”的节奏。

## 8. 怪物兵种系统

## 8.1 兵种大类

| 类型 | 特点 | 克制对象 | 被克制对象 |
|---|---|---|---|
| 小怪群 | 数量多、便宜 | 单体输出英雄 | 范围伤害英雄 |
| 坦克怪 | 血厚、挡路 | 高爆发低持续英雄 | 穿透、百分比伤害 |
| 远程怪 | 安全输出 | 近战英雄、短手英雄 | 突进英雄、飞剑流 |
| 刺客怪 | 快速贴脸 | 脆皮、远程流英雄 | 范围控制英雄 |
| 法术怪 | 异常状态 | 护盾流、回血流 | 高魔抗英雄 |
| 控制怪 | 减速、眩晕 | 高机动英雄 | 免控英雄 |
| 爆炸怪 | 死亡自爆 | 吸血流、召唤流英雄 | 高闪避英雄 |
| 禁疗怪 | 降低回血 | 吸血流英雄 | 高爆发英雄 |
| 破盾怪 | 专打护盾 | 护盾流英雄 | 无盾英雄收益低 |
| 精英怪 | 强力单位 | 中后期压制 | 消耗高，被集火 |

---

| 兵种 | 消耗 | 定位 | 技能 | 克制 |
|---|---:|---|---|---|
| 骷髅兵 | 10 | 基础小怪 | 近战攻击 | 单体英雄 |
| 毒虫 | 15 | 持续伤害 | 攻击附毒 | 护盾流、坦克流 |
| 小恶魔 | 20 | 高速骚扰 | 快速接近英雄 | 远程英雄 |
| 弓箭手 | 25 | 远程输出 | 远距离射击 | 近战英雄 |
| 石甲怪 | 40 | 坦克挡路 | 高血量低移速 | 直线输出英雄 |
| 爆裂虫 | 45 | 自爆单位 | 死亡爆炸 | 吸血流、召唤流 |
| 巫毒祭司 | 60 | 禁疗辅助 | 降低英雄回血 | 吸血流 |
| 破盾魔 | 70 | 破盾单位 | 对护盾造成额外伤害 | 护盾流 |
| 暗影刺客 | 80 | 突进爆发 | 闪现到英雄附近 | 脆皮流 |
| 巨锤精英 | 120 | 精英控制 | 范围击退、眩晕 | 高移速英雄 |

---

## 8.3 怪物属性字段

每个怪物配置建议包含以下字段：

| 字段 | 说明 |
|---|---|
| 怪物 ID | 唯一标识 |
| 怪物名称 | 显示名称 |
| 类型 | 小怪、远程、坦克、功能、精英、Boss |
| 生命值 | 承伤能力 |
| 攻击力 | 单次伤害 |
| 攻击频率 | 攻击速度 |
| 移动速度 | 接近英雄速度 |
| 攻击距离 | 近战或远程 |
| 能量消耗 | 召唤成本 |
| 召唤冷却 | 再次召唤间隔 |
| 场上上限 | 同类怪最大存在数量 |
| 经验价值 | 被英雄击杀后提供多少经验 |
| 克制标签 | 禁疗、破盾、减速、控制等 |
| 被克制标签 | 怕火、怕范围、怕穿透等 |
| 死亡效果 | 爆炸、毒雾、分裂等 |

## 9. 克制关系系统

## 9.1 核心克制逻辑

```text
英雄选择流派
↓
玩家观察英雄表现
↓
玩家判断英雄弱点
↓
玩家从士兵抽屉选择克制兵种
↓
玩家通过召唤位置和节奏制造压力
↓
```

## 10. 地图与障碍系统

## 10.1 地图设计目标

地图不是背景，而是玩家策略的一部分。

玩家可以利用地图：

1. 限制英雄移动。
2. 制造包围区域。
3. 保护远程怪。
4. 引导英雄进入陷阱。
5. 延长英雄绕路时间。
6. 给 Boss 创造输出空间。
7. 分割战场，降低英雄清怪效率。

## 11. 战斗节奏设计

## 11.1 单局时间结构

| 时间段 | 英雄状态 | 玩家策略 |
|---|---|---|
| 0 ~ 2 分钟 | 英雄弱，技能少 | 用小怪试探流派 |
| 2 ~ 5 分钟 | 英雄开始成型 | 根据流派选择克制怪 |
| 5 ~ 8 分钟 | 英雄清怪能力增强 | 使用地形、控制、远程组合 |
| 8 ~ 12 分钟 | 英雄进入强势期 | 召唤精英和高级怪 |
| 12 分钟后 | 英雄接近完全体 | 召唤 Boss，最终压制 |

## 12. 胜负条件

## 12.1 玩家胜利条件

满足任意一项即可胜利：

1. 英雄生命值归零。
2. 英雄被 Boss 击败。
3. 英雄被地图机制击杀。
4. 英雄崩溃值达到上限。
5. 英雄未能完成关卡目标。

## 12.2 玩家失败条件

满足任意一项即失败：

1. 英雄存活到倒计时结束。
2. 英雄击败最终 Boss。
3. 英雄摧毁所有召唤核心。
4. 英雄等级达到最终觉醒状态。
5. 英雄突破试炼门逃离地图。

## 20.3 平衡原则

1. 便宜怪不能无脑堆死英雄。
2. 强力怪不能放出来就必杀。
3. 英雄升级不能完全失控。
4. 玩家必须有反制空间。
5. 英雄强势时，玩家要获得更强工具。
6. 玩家错误召唤会变成“送经验”。
7. 每种英雄流派必须有明确弱点。
8. 召唤位置必须影响战局。
9. 地形布置必须有实际价值。
10. Boss 是决胜工具，不是必胜按钮。

## 21. 玩法案例

## 21.1 案例一：英雄走火焰范围流

### 英雄表现

- 周围持续火圈。
- 小怪靠近就死。
- 清怪速度很快。
- 低血小怪会快速变成英雄经验。

### 玩家应对

1. 停止大量召唤骷髅兵。
2. 攒能量召唤石甲怪。
3. 在远处召唤弓箭手。
4. 用墙体限制英雄靠近远程怪。
5. 召唤火抗精英怪顶住火焰。
6. 英雄血量下降后放暗影刺客收割。

---

## 21.2 案例二：英雄走吸血流

### 英雄表现

- 击杀怪物回血。
- 小怪越多，回血越快。
- 持久战很强。

### 玩家应对

1. 减少低血小怪，避免喂血。
2. 召唤禁疗祭司。
3. 用毒沼和减速区限制走位。
4. 召唤爆裂虫打瞬间伤害。
5. 在禁疗生效期间召唤精英怪集火。
6. 最后用腐毒女巫 Boss 收尾。

---

## 21.3 案例三：英雄走护盾流

### 英雄表现

- 定期获得护盾。
- 爆发伤害打不进去。
- 容错很高。

### 玩家应对

1. 等护盾出现后召唤破盾魔。
2. 用毒虫持续磨血。
3. 不在护盾期间浪费 Boss 技能。
4. 用控制怪拖住英雄。
5. 护盾消失瞬间召唤高爆发刺客。
6. 配合陷阱区完成击杀。

## 24. 原型验证重点

第一版重点验证以下问题：

1. 玩家看 AI 英雄自动战斗是否有趣。
2. 玩家是否能理解“我是反派 / 造物主”的目标。
3. 士兵抽屉召唤是否顺手。
4. 玩家是否能看懂英雄流派。
5. 克制关系是否明显。
6. 怪物放置位置是否有策略。
7. 地形布置是否真的影响战斗。
8. 玩家失败时是否知道为什么失败。
9. 玩家胜利时是否觉得是自己策略成功，而不是随机赢。
10. 召唤能量是否形成了有效决策压力。
11. 英雄成长是否会带来紧张感。
12. Boss 召唤是否有决胜爽感。

---

## 25. 与普通塔防的区别

本作需要避免变成普通塔防。

| 普通塔防 | 本作 |
|---|---|
| 敌人沿路线前进 | 英雄自由移动 |
| 玩家建塔防守 | 玩家召唤怪物进攻 |
| 怪物是被打对象 | 怪物是玩家的武器 |
| 目标是守住基地 | 目标是杀死英雄 |
| 关卡路线固定 | 玩家可改变战场环境 |
| 敌人不会成长 | 英雄会肉鸽成长 |
| 玩家防守 | 玩家主动围杀 |
| 核心是路线规划 | 核心是流派克制和压力节奏 |

---

## 26. 设计风险与解决方案

## 26.1 风险一：玩家感觉只是在刷怪

### 问题

如果召唤怪物没有克制关系，游戏会变成无脑点按钮。

### 解决方案

- 增加英雄流派识别。
- 增加明显克制标签。
- 错误召唤会给英雄送经验。
- 不同怪物对不同流派收益差异明显。
- 增加战斗提示：“英雄吸血过强，建议使用禁疗单位”。

---

## 26.2 风险二：玩家看不懂英雄为什么变强

### 问题

英雄自动升级，如果表现不清晰，玩家会觉得莫名其妙。

### 解决方案

- 右侧显示英雄已选卡牌。
- 英雄获得核心卡时弹出提示。
- 英雄流派图标发生变化。
- 技能特效明显改变。
- UI 提示当前英雄主要威胁。

---

## 26.3 风险三：太像塔防

### 问题

如果英雄路线固定，玩家只是造塔，就失去创新点。

### 解决方案

- 英雄必须自由移动。
- 怪物是主动追击单位。
- 玩家主要召唤怪物，而不是建塔。
- 地形是辅助，不是唯一核心。
- 英雄会根据战局改变移动策略。

---

## 26.4 风险四：AI 英雄太蠢或太强

### 问题

AI 太蠢会无聊，AI 太强会挫败。

### 解决方案

- 给 AI 设置难度档位。
- 低难度 AI 反应慢、卡牌选择普通。
- 高难度 AI 会优先选择强流派。
- AI 受到压制时有逃生逻辑。
- AI 不应完美操作，要保留可被设计击败的空间。

## 27. 后续扩展方向

## 27.1 多英雄挑战

后期可以加入双英雄或小队英雄。

例如：

- 战士 + 法师。
- 圣骑士 + 游侠。
- 召唤师 + 刺客。

玩家需要同时处理多个流派组合。

## 27.2 多造物主派系

玩家可以选择不同造物主阵营。

| 造物主 | 特点 |
|---|---|
| 骨王 | 擅长人海、复活、骨墙 |
| 毒母 | 擅长毒伤、禁疗、持续消耗 |

| 熔岩领主 | 擅长火焰、爆炸、高伤害 |
| 机械暴君 | 擅长炮塔、机关、精密控制 |
| 腐化森林 | 擅长藤蔓、减速、召唤巢穴 |

## 27.3 玩家自定义试炼

允许玩家自己编辑地图，配置怪物和规则，制作关卡给其他玩家挑战。

可包含：

- 地图编辑器。
- 怪物配置。
- 陷阱配置。
- 英雄配置。
- 时间限制。
- 胜负目标。
- 关卡分享码。

## 28. 总结

这款游戏的核心不是“造很多怪”，而是：

> 观察 AI 英雄的成长流派，利用地图、怪物、陷阱、精英和 Boss 形成克制组合，像设计一场死亡试炼一样，把英雄逼入绝境。

核心乐趣来自三点：

1. **反向身份爽感**：玩家不再是幸存者，而是要杀死幸存者的造物主。
2. **流派反制策略**：英雄越成长，玩家越需要调整针对策略。
3. **战场导演感**：玩家通过怪物、地形和时机设计一场英雄必死的局。
