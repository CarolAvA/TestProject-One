# 造物主技能系统策划案

## 1. 系统定位

造物主技能是玩家在战斗阶段直接干预战场的核心手段。

玩家不直接控制英雄，也不直接控制怪物移动，而是通过“召唤士兵 + 布置陷阱 + 释放造物主技能”来击败 AI 英雄。

造物主技能的特点：

1. 消耗造物主能量。
2. 有释放前摇和地面预警。
3. AI 英雄有机会躲避。
4. 技能不是必中，需要玩家预判。
5. 不同技能克制不同英雄流派。
6. 技能可以和怪物、障碍物、陷阱形成连招。
7. 强力技能有较长冷却，不能无脑 spam。

核心体验：

> 玩家像关卡 Boss 一样，用大范围技能、控制、诅咒、地形压制和怪物组合，设计一场让英雄难以逃脱的死亡试炼。

---

## 2. 技能系统核心规则

### 2.1 技能基础属性

每个造物主技能都需要配置以下属性：

| 属性 | 说明 |
|---|---|
| 技能名称 | 技能显示名称 |
| 技能类型 | 伤害、控制、诅咒、召唤、地形、强化 |
| 消耗能量 | 释放技能需要消耗的造物主能量 |
| 冷却时间 | 技能释放后的等待时间 |
| 释放范围 | 玩家可选择的释放区域 |
| 影响范围 | 技能实际生效区域 |
| 预警时间 | 技能落下前给 AI 英雄反应的时间 |
| 生效时间 | 技能真正造成效果的持续时间 |
| 命中判定 | 技能如何判断是否命中英雄 |
| 可躲避性 | AI 英雄是否能通过移动、闪避、位移躲开 |
| 克制标签 | 破盾、禁疗、减速、控制、打断、爆发等 |

---

### 2.2 释放流程

```text
玩家选择技能
↓
进入技能瞄准状态
↓
玩家点击或拖拽指定释放位置
↓
扣除能量，技能进入预警阶段
↓
地图上显示红圈 / 扇形 / 直线 / 路径预警
↓
AI 英雄根据危险等级尝试躲避
↓
预警结束，技能正式生效
↓
判定是否命中英雄 / 怪物 / 地形
↓
造成伤害、控制或其他效果
↓
技能进入冷却
```

## 3. 技能消耗系统

造物主技能和召唤士兵共用“造物主能量”，也可以独立使用“神力值”。  
推荐第一版先共用同一套资源，便于玩家理解。

### 3.1 共用能量方案

| 行为 | 消耗资源 |
|---|---|
| 召唤小怪 | 造物主能量 |
| 召唤精英 | 造物主能量 |
| 放置陷阱 | 造物主能量 |
| 释放技能 | 造物主能量 |
| 召唤 Boss | 大量造物主能量 |

优点：

1. 玩家需要在“召怪”和“放技能”之间取舍。
2. 技能不能无脑乱放。
3. 强化策略选择：是用怪物持续压制，还是攒能量打一波技能爆发。
4. 更符合“造物主调度全局资源”的感觉。

---

### 3.2 技能消耗档位

| 技能等级 | 消耗范围 | 示例 |
|---|---:|---|
| 小型技能 | 30~60 | 小范围落雷、短暂减速 |
| 中型技能 | 80~150 | 毒雾、地刺、黑暗冲击 |
| 大型技能 | 180~300 | 陨石、黑洞、全屏诅咒 |
| 终结技能 | 400~800 | 灭世审判、魔神之手 |

---

### 3.3 能量不足反馈

当玩家能量不足时，需要明确反馈：

| UI反馈 | 说明 |
|---|---|
| 技能按钮变灰 | 表示暂时不能释放 |
| 消耗数字变红 | 告诉玩家缺多少能量 |
| 点击后弹出提示 | “造物主能量不足” |
| 能量条轻微闪烁 | 强调资源不足 |
| 技能详情显示需求 | 当前 80 / 150 |

## 4. 技能预警系统

造物主技能必须有预警。  
预警是这个系统的核心，因为它决定了技能是否公平、是否有博弈感。

### 4.1 预警表现

| 技能形状 | 预警表现 |
|---|---|
| 圆形范围 | 地面出现红色圆圈，边缘闪烁 |
| 扇形范围 | 出现扇形危险区域 |
| 直线范围 | 地面出现长条红色路径 |
| 多段落点 | 多个小红圈依次出现 |
| 追踪技能 | 英雄脚下出现锁定标记 |
| 延迟爆炸 | 区域中心出现倒计时光效 |
| 持续地形 | 地面逐渐被腐化、燃烧、结冰 |

---

### 4.2 预警时间设计

| 技能类型 | 预警时间 | 设计目的 |
|---|---:|---|
| 小范围快速技能 | 0.4~0.8 秒 | 考验玩家预判 |
| 普通范围技能 | 1.0~1.5 秒 | AI 有明显反应时间 |
| 大范围高伤技能 | 1.8~3.0 秒 | 威力大，但容易躲 |
| 追踪型技能 | 1.2~2.0 秒 | 给英雄使用闪避的机会 |
| 全屏或半屏技能 | 3.0 秒以上 | 必须有安全区 |

---

### 4.3 预警强弱规则

技能越强，预警越明显。

| 技能强度 | 预警要求 |
|---|---|
| 低伤害 | 可短预警 |
| 高伤害 | 必须长预警 |
| 强控制 | 必须明显预警 |
| 大范围 | 必须显示安全方向 |
| 终结技 | 必须有非常清晰的倒计时和安全区 |

---

## 5. AI 英雄躲避逻辑

AI 英雄不是傻站着吃技能。  
当造物主技能进入预警阶段时，AI 会根据危险程度决定是否躲避。

### 5.1 AI 躲避判断流程

```text
检测到技能预警
↓
判断技能危险等级
↓
判断自己当前血量、护盾、移速、闪避技能冷却
↓
判断能否走出危险范围
↓
选择最优逃离方向
↓
如果走不出去，尝试使用闪避 / 位移 / 护盾
↓
如果无法躲避，选择承伤最小区域
```

---

### 5.2 危险等级判断

| 条件 | 危险等级 |
|---|---|
| 技能伤害低 | 低 |
| 技能带减速 | 中 |
| 技能带眩晕 | 高 |
| 技能可打断英雄回血 | 高 |
| 技能伤害足以致死 | 极高 |
| 技能命中后会被怪物包围 | 极高 |

---

### 5.3 AI 躲避优先级

AI 英雄面对技能预警时，按以下优先级行动：

1. 优先离开即将爆炸的范围。
2. 优先避开高伤害中心区域。
3. 优先朝怪物较少的方向移动。
4. 优先避开墙体死角。
5. 血量低时优先保命。
6. 护盾充足时可以选择硬吃低伤技能。
7. 技能冷却好时，可以使用闪避离开危险区。
8. 被减速或包围时，可能无法完全躲开。

---

### 5.4 AI 躲避成功率影响因素

| 因素 | 影响 |
|---|---|
| 英雄移速 | 移速越高，越容易躲开 |
| 当前被包围程度 | 怪物越多，越难躲 |
| 障碍物位置 | 路线被堵，容易被命中 |
| 技能预警时间 | 预警越长，越容易躲 |
| 技能范围 | 范围越大，越难躲 |
| 英雄闪避技能 | 有闪避时更容易逃脱 |
| 控制状态 | 被减速、眩晕时难以躲避 |
| 玩家释放位置 | 预判越准，命中率越高 |

## 6. 玩家技能释放操作

### 6.1 基础操作

| 操作 | 效果 |
|---|---|
| 点击技能按钮 | 进入技能瞄准状态 |
| 点击地图 | 在目标位置释放 |
| 按住技能按钮 | 显示技能范围和说明 |
| 拖拽技能按钮到地图 | 快速释放 |
| 双击技能按钮 | 对英雄当前位置快速释放 |
| 点击取消区域 | 取消释放，不消耗能量 |

---

### 6.2 移动端推荐操作

移动端要避免复杂操作，推荐两种方式：

#### 方式一：点选释放

1. 点击技能按钮。
2. 地图显示可释放范围。
3. 点击目标区域。
4. 技能开始预警。
5. 预警结束后生效。

适合：陨石、毒雾、黑洞、地刺。

#### 方式二：拖拽释放

1. 按住技能按钮。
2. 拖到地图目标点。
3. 松手释放。
4. 技能进入预警。

适合：快速操作、战斗节奏快的技能。

---

### 6.3 PC 端推荐操作

| 操作 | 效果 |
|---|---|
| 数字键 1~6 | 选择技能 |
| 鼠标左键 | 确认释放 |
| 鼠标右键 | 取消释放 |
| Shift + 技能键 | 快速对英雄释放 |
| 鼠标悬停 | 显示技能详情 |
| Alt | 显示所有技能预警范围 |


## 8. 战场预警 UI

### 8.1 地面预警层级

战场上的技能预警需要清晰，但不能太乱。

| 层级 | 显示内容 |
|---|---|
| 第一层 | 地面范围形状 |
| 第二层 | 边缘高亮 |
| 第三层 | 中心倒计时 |
| 第四层 | 技能落点图标 |
| 第五层 | 安全方向提示，仅大型技能使用 |

---

### 8.2 预警颜色建议

| 危险程度 | 表现 |
|---|---|
| 低危险 | 淡红色范围 |
| 中危险 | 红色范围 + 边缘闪烁 |
| 高危险 | 深红色范围 + 倒计时 |
| 极高危险 | 红黑色范围 + 屏幕轻微震动 |
| 友方强化 | 紫色或蓝色范围 |
| 地形变化 | 对应元素颜色，例如毒绿、冰蓝、火橙 |

---

### 8.3 命中反馈

技能命中英雄时要有明确反馈：

| 命中情况 | 表现 |
|---|---|
| 完全命中 | 大数字、震屏、英雄受击动作 |
| 边缘命中 | 较小伤害数字 |
| 被护盾抵消 | 显示“护盾吸收” |
| 被闪避躲开 | 显示“闪避” |
| AI 走出范围 | 显示“未命中”或不显示 |
| 控制命中 | 英雄头顶显示控制图标 |
| 禁疗命中 | 英雄血条旁显示禁疗标记 |

## 9. 技能类型设计

### 9.1 伤害型技能

用于直接压低英雄血量。

| 技能 | 消耗 | 冷却 | 预警 | 效果 |
|---|---:|---:|---:|---|
| 落雷 | 50 | 6 秒 | 0.6 秒 | 小范围高伤害 |
| 地刺 | 80 | 10 秒 | 1 秒 | 直线伤害并短暂击退 |
| 陨石 | 180 | 22 秒 | 2 秒 | 大范围高伤害 |
| 魔焰爆破 | 140 | 18 秒 | 1.5 秒 | 中范围火焰伤害 |

设计重点：

- 伤害越高，预警越长。
- 小技能适合补刀。
- 大技能适合配合控制或障碍物使用。

---

### 9.2 控制型技能

用于限制英雄走位，配合怪物输出。

| 技能 | 消耗 | 冷却 | 预警 | 效果 |
|---|---:|---:|---:|---|
| 暗影锁链 | 100 | 16 秒 | 1 秒 | 命中后短暂定身 |
| 冰霜领域 | 120 | 20 秒 | 1.2 秒 | 区域减速 |
| 重力井 | 160 | 25 秒 | 1.5 秒 | 将英雄向中心拉扯 |
| 恐惧尖啸 | 130 | 22 秒 | 1 秒 | 让英雄短暂失控逃跑 |

设计重点：

- 控制技能本身伤害低。
- 主要用于让怪物和其他技能命中。
- 对高机动流派有明显克制。

---

### 9.3 诅咒型技能

用于削弱英雄状态。

| 技能 | 消耗 | 冷却 | 预警 | 效果 |
|---|---:|---:|---:|---|
| 禁疗诅咒 | 120 | 18 秒 | 1.2 秒 | 降低回血和吸血 |
| 破盾烙印 | 110 | 16 秒 | 1 秒 | 降低护盾强度 |
| 虚弱诅咒 | 100 | 20 秒 | 1 秒 | 降低攻击力 |
| 致盲黑雾 | 90 | 15 秒 | 1 秒 | 降低索敌范围 |

设计重点：

- 不一定直接杀人，但可以破坏英雄流派。
- 适合针对吸血、护盾、远程、召唤流。

---

### 9.4 地形型技能

用于改变战场环境。

| 技能 | 消耗 | 冷却 | 预警 | 效果 |
|---|---:|---:|---:|---|
| 腐化沼泽 | 130 | 24 秒 | 1.5 秒 | 生成减速毒地 |
| 骨墙升起 | 150 | 28 秒 | 1 秒 | 临时生成阻挡墙 |
| 熔岩裂缝 | 180 | 30 秒 | 2 秒 | 地面持续喷火 |
| 虚空裂隙 | 200 | 35 秒 | 1.5 秒 | 改变怪物出生点 |

设计重点：

- 地形技能适合制造连招。
- 不能完全封死英雄路线。
- 临时障碍需要有持续时间和可破坏规则。

---

### 9.5 强化型技能

用于增强玩家召唤的怪物。

| 技能 | 消耗 | 冷却 | 预警 | 效果 |
|---|---:|---:|---:|---|
| 狂暴号令 | 100 | 20 秒 | 0.5 秒 | 区域内怪物攻速提高 |
| 血肉强化 | 130 | 25 秒 | 0.8 秒 | 怪物生命提高 |
| 暗影加速 | 90 | 18 秒 | 0.5 秒 | 怪物移速提高 |
| 亡者复苏 | 160 | 30 秒 | 1 秒 | 复活区域内部分死亡怪物 |

设计重点：

- 强化技能主要作用于怪物。
- 适合在英雄被包围时释放。
- 可以和召唤波次形成爆发。

---

### 9.6 终结型技能

用于后期决胜。

| 技能 | 消耗 | 冷却 | 预警 | 效果 |
|---|---:|---:|---:|---|
| 魔神之手 | 500 | 60 秒 | 2.5 秒 | 巨手拍击大范围区域 |
| 灭世陨星 | 700 | 90 秒 | 3 秒 | 超大范围爆炸，有安全区 |
| 深渊吞噬 | 650 | 80 秒 | 2.5 秒 | 黑洞拉扯并持续伤害 |
| 万魂审判 | 800 | 100 秒 | 3 秒 | 根据英雄已击杀怪物数量造成额外伤害 |

设计重点：

- 威力强，但非常容易被预判躲避。
- 最好配合控制、减速、障碍物、怪物包围使用。
- 不能设计成点击就赢。

## 10. 技能连招设计

造物主技能最大的乐趣来自组合。

### 10.1 基础连招

| 连招 | 逻辑 |
|---|---|
| 减速区域 + 陨石 | 先降低英雄移动速度，再用大范围伤害命中 |

| 禁疗诅咒 + 爆炸怪 | 禁止回血后，用爆发伤害击杀 |
| 重力井 + 小怪群 | 把英雄拉进怪物群中 |
| 黑雾致盲 + 远程怪 | 降低英雄索敌，让远程怪输出更久 |
| 破盾烙印 + 精英怪 | 先削护盾，再让精英打爆发 |
| 腐化沼泽 + 毒虫 | 毒伤叠加，压制回血流 |
| 暗影锁链 + 魔神之手 | 控住英雄后释放终结技 |

## 12. 技能命中判定

### 12.1 圆形技能

判定规则：

```text
英雄中心点在技能半径内 → 完全命中
英雄碰撞体边缘进入范围 → 边缘命中
英雄离开范围 → 未命中
```

### 12.2 直线技能

判定规则：

```text
技能从起点向终点生成伤害带
英雄碰撞体和伤害带重叠 → 命中
英雄使用闪避穿过 → 根据无敌帧判断是否免疫
```

### 12.3 持续区域技能

判定规则：

```text
英雄进入区域 → 每隔固定时间受到一次效果
英雄离开区域 → 停止受到效果
英雄再次进入 → 重新生效
```

### 12.4 追踪技能

判定规则：

```text
预警时锁定英雄当前位置
技能落下时打击锁定位置
如果英雄移动离开 → 未命中
如果英雄被控制无法离开 → 命中
```

## 13. 技能释放限制

为了防止玩家无脑封锁英雄，需要以下限制：

| 限制 | 说明 |
|---|---|
| 不能连续释放同类控制 | 防止无限控 |
| 大技能共享全局冷却 | 防止连续秒杀 |
| 地形技能不能完全封死路线 | 必须留逃生空间 |
| 终结技一局次数有限 | 保持稀缺性 |
| 同一区域连续释放有衰减 | 防止固定点压制 |
| 英雄有抗性成长 | 经常被控制后会获得短暂抗性 |
| 技能有能量成本 | 放技能会影响召怪节奏 |

## 14. 英雄反制技能机制

AI 英雄除了走位，还可以拥有反制技能。

### 14.1 英雄反制方式

| 反制方式 | 说明 |
|---|---|
| 闪避 | 快速离开预警区 |
| 护盾 | 硬吃一次伤害 |
| 位移 | 冲刺到安全位置 |
| 净化 | 清除诅咒状态 |
| 打断 | 攻击施法核心，取消技能 |
| 加速 | 短时间提高移速 |
| 减伤 | 降低即将受到的伤害 |

---

### 14.2 英雄是否反制的判断

AI 会根据收益判断是否使用反制：

| 情况 | 行为 |
|---|---|
| 技能伤害很低 | 不浪费闪避 |
| 当前血量很低 | 优先闪避 |
| 技能带禁疗或控制 | 优先躲避 |
| 护盾充足 | 可以硬吃 |
| 被怪物包围 | 优先闪避出包围 |
| 闪避冷却中 | 寻找最近安全区 |
| 终结技预警 | 必定尝试逃离 |

## 16. UI 重点信息优先级

战斗中信息很多，必须分优先级。

### 16.1 玩家最需要看的信息

优先级从高到低：

1. 当前造物主能量。
2. 哪些技能可以释放。
3. 技能冷却时间。
4. 英雄当前位置和移动方向。
5. 技能预警范围。
6. 英雄是否已经被减速、禁疗、破盾。
7. 技能是否命中。
8. 当前英雄流派。
9. 怪物和陷阱状态。

---

### 16.2 UI 避免的问题

| 问题 | 解决方式 |
|---|---|
| 技能按钮太多 | 技能栏最多显示 4~6 个，其他放二级抽屉 |
| 预警特效太乱 | 只强化当前技能和高危技能 |
| 玩家看不懂没命中原因 | 显示“闪避”“护盾吸收”“未命中” |
| 能量消耗不明显 | 每次释放时能量条扣减动画 |
| 冷却不明显 | 圆形遮罩 + 数字倒计时 |
| 克制关系不明显 | 技能按钮显示克制标签 |

## 19. 设计结论

造物主技能不是“免费必中伤害”，而是一套围绕预判、消耗、预警和 AI 躲避建立的博弈系统。

玩家真正的乐趣不是单点技能秒杀英雄，而是：

> 先用怪物和地形限制英雄走位，再用造物主技能制造危险区域，逼英雄交闪避、进死角、吃控制，最后通过克制技能和怪物完成击杀。
