# AI 控制角色成长 BD 系统方案  
## 反幸存者玩法用：AI 英雄伤害能力 / 音乐节奏 / 弹幕成长完整设计

---

# 1. 系统定位

本系统用于“反幸存者”玩法中的 AI 控制角色。

玩家不直接控制这个角色，而是扮演造物主，通过召唤怪物、放置障碍、释放技能来试图击败 AI 角色。

AI 角色本身拥有类似《吸血鬼幸存者》的成长 BD，但这套 BD 不只是简单提升伤害，而是和“音效播放器 / 节奏预警器 / 弹幕释放节拍”绑定。

核心设计目标：

> AI 角色的每一次攻击，都应该像音乐里的一个节拍。  
> AI 的成长 BD 会改变弹幕形态、音色层次、节奏密度、音轨复杂度和视觉特效。  
> 玩家可以通过屏幕下方的节奏播放器，提前判断 AI 正在形成什么流派，并选择对应的反制手段。

---

# 2. 核心玩法概念

AI 初始只有一个最基础能力：

> 按固定节奏，向目标发射一枚基础子弹。

随着局内升级，AI 会获得不同 BD：

1. 基础属性 BD。
2. 额外弹体 BD。
3. 路径区域 BD。
4. AOE 音波 BD。
5. 节奏音轨 BD。
6. 元素状态 BD。
7. 弹幕形态 BD。
8. 音乐表现 BD。
9. 终局流派 BD。

最终 AI 会从“单发子弹”成长为一个具有完整音乐节奏、复合弹幕、多元素状态和强烈视觉表现的自动战斗角色。

---

# 3. 设计关键词

| 关键词 | 说明 |
|---|---|
| 节奏 | 所有伤害释放都受节奏器控制 |
| 音色 | 不同弹体拥有不同音效 |
| 音轨 | 高音、中音、低音分别表现不同攻击类型 |
| 间隔 | 多种弹体不会完全同时释放，而是按节拍错开发射 |
| 可读性 | 玩家能通过音效和 UI 预判攻击 |
| 成长感 | AI 从简单单发，成长为复杂乐章 |
| 反制性 | 每个 BD 都要有造物主可反制的窗口 |
| 表演性 | 攻击要有音乐、特效、弹幕的同步爽感 |

---

# 4. 基础攻击规则

## 4.1 初始能力：单音灵弹

AI 初始技能：

| 项目 | 内容 |
|---|---|
| 名称 | 单音灵弹 |
| 攻击方式 | 按节奏发射 1 枚基础子弹 |
| 默认节奏间隔 | 1.2 秒一次 |
| 默认弹体速度 | 中等 |
| 默认伤害 | 100% |
| 默认发射数量 | 1 |
| 默认音效 | 清脆短音，类似“叮” |
| 默认音轨 | 中音轨 |
| 默认特效 | 小型白蓝色光弹 |

基础表现：

```text
中音轨：━━●━━━━●━━━━●━━━━●━━
音效：叮    叮    叮    叮
弹幕：每个节拍点发射一枚基础子弹
```

---

## 4.2 攻击释放原则

所有伤害能力都不应该随机乱放，而应该受一个统一节奏器控制。

正确逻辑：

```text
AI 当前攻击循环
↓
节奏器推进
↓
到达节拍点
↓
根据当前 BD 配置选择本次触发的伤害模块
↓
播放对应音效
↓
生成对应弹幕 / AOE / 路径区域
↓
UI 音轨节拍点闪烁
```

# 5. 核心数值属性 BD

基础属性 BD 是所有流派都可以拿的通用成长。

---

## 5.1 弹体飞行速度

### 设计说明

提升所有弹体的飞行速度。

| 等级 | 效果 |
|---|---|
| Lv.1 | 弹体飞行速度 +10% |
| Lv.2 | 弹体飞行速度 +20% |
| Lv.3 | 弹体飞行速度 +35% |
| Lv.4 | 弹体飞行速度 +50% |
| Lv.5 | 弹体飞行速度 +70%，弹体拖尾变长 |

### 音效表现

速度越高，音效越尖锐，尾音越短。

| 等级 | 声音 |
|---|---|
| 低等级 | “咻” |
| 中等级 | “嗖” |
| 高等级 | “嘶——”短促破空 |

### 特效表现

- 弹体拖尾变长。
- 高等级出现细线型光痕。
- 弹体边缘增加空气切割波纹。
- 速度达到高等级后，屏幕上出现短暂残影。

### 玩家感知

玩家会感觉 AI 的弹幕更难躲，怪物更容易被命中。

### 造物主反制

- 放置高血量怪物吸收弹幕。
- 放置障碍物阻挡直线弹体。
- 召唤分散型小怪，降低高速弹的命中收益。

---

## 5.2 节奏速度

### 设计说明

降低所有伤害释放的节奏间隔，也就是整体攻击 CD 变短。

| 等级 | 效果 |
|---|---|
| Lv.1 | 节奏间隔 -8% |
| Lv.2 | 节奏间隔 -15% |
| Lv.3 | 节奏间隔 -25% |
| Lv.4 | 节奏间隔 -35% |
| Lv.5 | 节奏间隔 -45%，进入快节奏音轨表现 |

原始节奏：

```text
━━●━━━━●━━━━●━━━━●━━
```

提升后：

```text
━━●━━●━━●━━●━━●━━●━━
```

### 音效表现

- 节拍更密。
- UI 进度器推进更快。
- 音效从“单音”变成“连续短音”。
- 高等级会出现鼓点感。

### 特效表现

- AI 身上出现随节拍跳动的光圈。
- 每次发射时脚下出现脉冲。
- 高等级节奏会让音效播放器边框轻微震动。

### 玩家感知

玩家会明显感觉 AI 输出频率变高，召唤小怪更容易被清掉。

### 造物主反制

- 召唤坦克怪扛节奏。
- 等 AI 一轮节奏结束后再召唤脆皮怪。
- 用造物主技能打断或限制 AI 节奏。

---

## 5.3 伤害提升

### 设计说明

提升 AI 所有伤害。

| 等级 | 效果 |
|---|---|
| Lv.1 | 所有伤害 +10% |
| Lv.2 | 所有伤害 +20% |
| Lv.3 | 所有伤害 +35% |
| Lv.4 | 所有伤害 +50% |
| Lv.5 | 所有伤害 +70%，重击节拍点获得额外闪光 |

### 音效表现

伤害越高，声音越厚，冲击感越强。

| 等级 | 声音变化 |
|---|---|
| 低 | 清脆轻音 |
| 中 | 带轻微冲击 |
| 高 | 命中时有“砰”的厚重感 |

### 特效表现

- 命中闪光变强。
- 伤害数字变大。
- 弹体核心亮度提高。
- 高等级命中时出现小范围震波。

### 玩家感知

造物主召唤出来的单位死亡更快，需要更注意克制。

---

## 5.4 发射数量

### 设计说明

每次释放伤害时，额外发射弹体。

| 等级 | 效果 |
|---|---|
| Lv.1 | 每次额外发射 +1 |
| Lv.2 | 每次额外发射 +2 |
| Lv.3 | 每次额外发射 +3 |
| Lv.4 | 每次额外发射 +4 |
| Lv.5 | 每次发射变为小型扇形阵列 |

### 发射方式

不是所有弹体叠在一起，而是形成可读性好的角度。

```text
Lv.1：1 发
Lv.2：3 发小扇形
Lv.3：5 发中扇形
Lv.4：7 发大扇形
Lv.5：9 发扇形，中心弹增强
```

### 音效表现

- 单发：一个音点。
- 多发：一个主音 + 多个轻音叠加。
- 高等级：类似拨弦扫过，“叮啷”。

### 特效表现

- 多条弹道轨迹。
- 扇形发射光线。
- 高等级每个弹体有微弱音符尾迹。

### 玩家感知

AI 清小怪能力变强，但对高血量单体的提升不如纯伤害 BD。

### 造物主反制

- 用高血量精英怪。
- 用散布范围更大的怪物。
- 从 AI 背后或侧后方召唤怪物，避开扇形正面。

# 7. 额外弹体 BD 系统

额外弹体不是和基础弹体同时释放，而是在节奏器内插入“额外音符”。

规则：

```text
基础弹体节拍点：●
额外弹体节拍点：○
```

示例：

```text
原始：
━━●━━━━●━━━━●━━━━

加入额外弹体后：
━━●━○━━●━○━━●━○━━
```

这样可以保证：

1. 音乐层次更丰富。
2. 玩家能听出不同弹体类型。
3. 弹幕不至于瞬间堆成一团。
4. UI 播放器能清楚显示不同音色。

# 26. 节奏音轨 BD 系统

这部分是音乐表现的核心，可以让 AI 的成长不仅是数值变化，而是变成“从简单节拍变成复杂乐曲”。

---

## 26.1 音轨分类

| 音轨 | 代表能力 |
|---|---|
| 高音轨 | 高频弹幕、闪电、冰晶、音刃 |
| 中音轨 | 基础弹、火球、普通连射、扇形攻击 |
| 低音轨 | 音爆、重鼓弹、震地、大招 |
| 持续层 | 毒雾、燃烧、冰霜区域 |
| 回响层 | 延迟伤害、命中回响、二段爆发 |

---

## 26.2 节奏 BD 一：切分节奏

### 效果

额外弹体会插入到主节拍之间。

```text
普通：
━━●━━━━●━━━━●━━━━

切分：
━━●━○━━●━○━━●━○━━
```

### 玩法意义

- AI 弹幕间隔更难被抓空档。
- 玩家看到 UI 后能判断攻击中间还有额外弹体。
- 很适合闪电弹、冰冻弹、毒气弹。

### 声音表现

主音后追加一个轻音。

```text
叮-哒，叮-哒，叮-哒
```

---

## 26.3 节奏 BD 二：三连音

### 效果

每隔数次攻击，会在短时间内连续释放 3 个快速节拍。

```text
━━●━━━━●━━●●●━━━━●━━
```

### 适合能力

- 基础弹。
- 闪电弹。
- 音刃弹。
- 火焰弹。

### 声音表现

```text
叮  叮  叮叮叮  叮
```

---

## 26.4 节奏 BD 三：重拍强化

### 效果

每第 4 次节拍变为重拍，伤害和特效提高。

```text
●  ●  ●  ◎
```

### 适合能力

- 火焰弹。
- 重鼓弹。
- 音爆。
- 节拍震地。

### 声音表现

前三次轻音，第四次低音重击。

```text
叮 叮 叮 咚！
```

---

## 26.5 节奏 BD 四：回声节拍

### 效果

每次主伤害后，延迟 0.4 秒追加一次低伤回响。

```text
●  ○    ●  ○    ●  ○
```

### 适合能力

- 毒气弹。
- 音波裂隙。
- 命中回响。
- 和弦爆发。

### 声音表现

```text
咚……嗡
咚……嗡
```

---

## 26.6 节奏 BD 五：渐强

### 效果

连续攻击不中断时，节奏逐渐加快，伤害逐渐提高。  
如果 AI 被打断、减速或进入躲避行为，则渐强层数下降。

| 层数 | 效果 |
|---|---|
| 1 层 | 正常 |
| 2 层 | 节奏 +5%，伤害 +5% |
| 3 层 | 节奏 +10%，伤害 +10% |
| 4 层 | 节奏 +15%，伤害 +15% |
| 5 层 | 触发一次强力和弦爆发 |

### 声音表现

音乐越来越密、音量略微提高、节拍点越来越亮。

### 玩法意义

玩家如果不干扰 AI，AI 会越打越强。  
造物主需要用控制、障碍、技能去打断 AI 的节奏。

---

## 26.7 节奏 BD 六：弱拍陷阱

### 效果

AI 在普通弱拍时释放小伤害，在强拍时释放控制或范围伤害。

```text
弱  弱  弱  强
●   ●   ●   ◎
```

### 玩法意义

玩家可以通过 UI 看到强拍快到了，提前避免召唤小怪或准备反制。

---

## 26.8 节奏 BD 七：复调音轨

### 效果

AI 同时拥有两条不同节奏的攻击线。  
例如基础弹维持中音轨，毒伤维持低音持续层。

```text
中音：━━●━━━━●━━━━●━━━━
低音：━━━━○━━○━━○━━○━━
```

### 玩法意义

AI 不再只是单一攻击，而是主攻击 + 持续伤害叠加。

### 平衡限制

复调音轨不能无限叠加。  
建议最多同时 3 条主动音轨，否则 UI 和战斗都会过乱。

---

## 26.9 节奏 BD 八：终曲段落

### 效果

AI 血量低于一定比例，或者战斗时间进入后期，会进入终曲段落。

| 条件 | 效果 |
|---|---|
| 血量低于 40% | 音轨亮度提高 |
| 血量低于 25% | 节奏速度提高 |
| 血量低于 10% | 解锁一次终曲大招 |

### 声音表现

- 底层鼓点增强。
- 主旋律更密。
- 大招前有明显低音蓄力。

### 特效表现

- AI 身后出现音符光环。
- 弹幕颜色更鲜明。
- UI 播放器边缘出现终曲边框。

- 召唤专门克制 AI 当前主音轨的怪。
- 不要给 AI 太多持续输出空间。

# 32. AI 选 BD 逻辑

AI 不是随机乱选卡，应该有构筑倾向。

---

## 32.1 AI 初始权重

每局 AI 可以随机获得一个隐藏倾向。

| 倾向 | 偏好 |
|---|---|
| 高频倾向 | 节奏速度、高音弹体、闪电、音刃 |
| 范围倾向 | 火焰、AOE、音爆、路径区域 |
| 控制倾向 | 冰冻、减速、节奏稳定 |
| 消耗倾向 | 毒气、路径毒雾、回响 |
| 重击倾向 | 低音、重鼓、震地、伤害 |
| 均衡倾向 | 多元素、和弦、复调 |

---

## 32.2 AI 选择规则

AI 升级时出现 3 张 BD 卡。  
AI 根据当前局势和隐藏倾向评分。

评分因素：

| 因素 | 说明 |
|---|---|
| 当前流派倾向 | 和已有 BD 越匹配分越高 |
| 当前敌人类型 | 场上小怪多则偏向范围 |
| 当前血量 | 血量低时偏向控制、击退、AOE |
| 玩家召唤方式 | 玩家喜欢堆怪，则偏向火焰/雷电 |
| 当前短板 | 缺范围补范围，缺控制补控制 |
| 稀有度 | 高稀有 BD 有额外权重 |

---

## 32.3 AI 选卡示例

场上大量小怪：

```text
火焰弹 +30 分
音爆 +25 分
闪电弹 +20 分
单体伤害 +5 分
```

场上高速刺客怪多：

```text
冰冻弹 +30 分
AOE 音爆 +20 分
节奏稳定 +15 分
毒气弹 +5 分
```

场上高血精英怪多：

```text
毒气弹 +30 分
伤害提升 +25 分
破甲音波 +20 分
火焰小怪清场 +5 分
```

# 34. 战场特效层级规范

为了避免后期弹幕太乱，需要设定特效层级。

---

## 34.1 特效优先级

| 优先级 | 内容 |
|---|---|
| 最高 | 造物主技能预警、Boss 技能预警 |
| 高 | AI 大招、低音重击、和弦爆发 |
| 中 | 元素弹体、AOE 音波 |
| 低 | 毒伤跳字、持续区域小火苗 |
| 最低 | 装饰音符、细碎粒子 |

---

## 34.2 可读性规则

1. 预警范围必须比装饰特效更清楚。
2. 弹体颜色要和元素一致。
3. 音轨 UI 的节拍点要和战场弹幕一一对应。
4. 持续区域透明度不能太高，避免遮挡怪物。
5. 大招音效不能和普通弹体音效混在一起。
6. 高频音效要柔和，不能真实刺耳。
7. 多元素流派要控制颜色数量，优先显示主元素。

# 35. 数值平衡建议

## 35.1 基础数值方向

| 属性 | 初始值建议 |
|---|---:|
| 基础节奏间隔 | 1.2 秒 |
| 基础弹体伤害 | 100 |
| 基础弹体速度 | 8 米/秒 |
| 基础弹体数量 | 1 |
| 基础弹体寿命 | 3 秒 |
| 基础暴击率 | 0% |
| 基础穿透 | 0 |
| 基础 AOE 半径 | 1.5 米 |
| 基础异常状态时长 | 0.5~3 秒 |

---

## 35.2 成长上限建议

| 属性 | 建议软上限 |
|---|---:|
| 节奏间隔降低 | 最多 -60% |
| 发射数量 | 最多 +8 |
| 弹体速度 | 最多 +100% |
| 伤害提升 | 最多 +150% |
| AOE 范围 | 最多 +120% |
| 状态持续时间 | 最多 +100% |
| 穿透次数 | 最多 6 |
| 弹射次数 | 最多 8 |
| 同时主动音轨 | 最多 3 条 |
| 持续区域层数 | 最多 3 层主要显示 |

---

## 35.3 强度控制原则

1. 节奏速度和发射数量不能无限叠。
2. 额外弹体越多，单个弹体伤害要有折减。
3. 控制类状态对精英和 Boss 要有衰减。
4. 持续区域不能覆盖全屏太久。
5. 多元素和弦强，但触发频率要低。
6. 终曲能力强，但必须有明显预警。
7. 玩家必须能通过 UI 看懂 AI 当前危险来源。

## 39.3 伪代码逻辑

```text
OnRhythmTick(time):
    events = RhythmTimeline.GetEventsAt(time)

    for event in events:
        AudioSystem.Play(event.AudioCue)
        RhythmUI.Flash(event.Track, event.BeatType)
        VFXSystem.Play(event.VisualCue)

        if event.HasProjectile:
            ProjectileSystem.Spawn(event.ProjectileData)

        if event.HasAOE:
            AOESystem.Spawn(event.AOEData)

        if event.HasPathArea:
            PathAreaSystem.AttachToProjectile(event.PathAreaData)

        if event.HasStatusEffect:
            StatusSystem.Apply(event.StatusData)
```

# 40. 最终体验目标

AI 角色的成长应该从这样：

```text
叮……叮……叮……
单发基础子弹
```

逐渐成长为：

```text
叮-滋-咚，叮叮叮，轰——，咔，噗嗤……
高音闪电切分
中音火焰爆破
低音音波震地
毒雾路径持续回响
冰霜弹冻结怪物
最终和弦爆发清场
```

玩家看到的是：

1. 下方节奏播放器越来越复杂。
2. AI 弹幕从单发变成多音轨。
3. 不同元素有不同声音。
4. 不同流派有不同音乐节奏。
5. 玩家可以通过音轨判断 AI 的构筑方向。
6. 玩家需要用造物主怪物、障碍和技能进行针对性反制。

# 41. 一句话总结

这套 AI 成长 BD 的核心不是单纯堆弹幕，而是：

> 让 AI 的每个伤害能力都变成音乐节奏中的一个声部。  
> 基础子弹是主旋律，额外弹体是副旋律，路径区域是持续音，AOE 音波是低音鼓点，终局流派是完整乐章。  
> 玩家既是在对抗一个自动成长的幸存者角色，也是在阅读一首越来越危险的战斗音乐。
