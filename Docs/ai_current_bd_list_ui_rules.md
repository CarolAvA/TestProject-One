# AI角色当前BD列表UI规则文档

## 1. 系统定位

在屏幕下方的 **AI音效进度条下面** 增加一个「当前BD列表」。

作用是让玩家随时看清楚：

1. AI角色当前拿了哪些BD。
2. 每个BD属于什么类型。
3. 当前BD等级是多少。
4. 这个BD对AI弹幕、节奏、伤害、特效有什么影响。
5. 鼠标悬停后显示详细Tips，方便玩家判断该怎么反制。

---

## 2. UI总体布局

### 2.1 屏幕下方结构

```text
┌──────────────────────────────────────┐
│ AI音效节奏进度条                      │
│ 低音轨 / 中音轨 / 高音轨 / 当前技能节奏 │
├──────────────────────────────────────┤
│ AI当前BD列表                          │
│ [伤害↑] [节奏+] [火焰弹] [毒气路径] [音波AOE] ... │
├──────────────────────────────────────┤
│ 玩家士兵抽屉 / 召唤栏                  │
│ [骷髅兵] [毒虫] [弓箭手] [精英] ...     │
└──────────────────────────────────────┘
```

推荐顺序：

```text
战场区域
↓
AI音效进度条
↓
AI当前BD列表
↓
造物主士兵抽屉
```

这样玩家看AI当前攻击节奏时，可以顺手看到AI当前成长方向。

## 3. 当前BD列表显示规则

### 3.1 基础显示内容

每个BD用一个Icon表示。

单个BD Icon需要显示：

| 显示内容 | 说明 |
|---|---|
| BD图标 | 表示BD类型 |
| 等级数字 | 例如 Lv.1 / Lv.2 / Lv.3 |
| 品质边框 | 普通、稀有、史诗、核心 |
| 类型角标 | 基础属性、弹体、路径、AOE、节奏、特殊 |
| 新获得闪光 | 刚获得时播放一次闪光 |
| 升级箭头 | BD升级时显示上箭头 |
| 层数标记 | 可叠加BD显示层数 |
| 禁用 / 被克制状态 | 如果该BD当前被玩家技能削弱，需要显示暗化或裂纹 |

---

### 3.2 Icon推荐尺寸

| 平台 | Icon尺寸 |
|---|---:|
| PC | 42 × 42 |
| 移动端 | 48 × 48 |
| 小屏模式 | 36 × 36 |

BD列表高度建议：

```text
PC：56 px ~ 72 px
移动端：64 px ~ 84 px
```

---

### 3.3 BD数量过多时的显示规则

一局中AI可能拿很多BD，不能无限横向堆满。

推荐规则：

| 当前BD数量 | 显示方式 |
|---|---|
| 1~8个 | 全部直接显示 |
| 9~14个 | 横向滚动列表 |
| 15个以上 | 分类折叠 + 横向滚动 |
| 核心BD | 永远优先显示在最左侧 |
| 最近获得BD | 短时间置顶显示3秒 |
| 同类可叠加BD | 合并成一个Icon，显示等级 / 层数 |

## 4. BD分类显示规则

| 分类 | 颜色 / 角标 | 示例 |
|---|---|---|
| 基础属性BD | 白色小箭头 | 伤害、飞行速度、节奏速度 |

| 路径类BD | 绿色轨迹角标 | 毒气路径、冰冻路径、燃烧路径 |

| 节奏类BD | 黄色音符角标 | 节拍加速、双拍、延迟重音 |
| 特殊机制BD | 红色星标 | 弹幕分裂、元素融合、终曲爆发 |

### 4.2 排序规则

```text

↓
弹体类BD
↓
路径类BD
↓
AOE类BD
↓
节奏类BD
↓
基础属性BD

```

如果是刚获得的BD，可以临时移动到最左侧并闪烁，3秒后回到正常排序。

## 6. 鼠标悬停Tips规则

### 6.1 触发方式

| 操作 | 效果 |
|---|---|
| 鼠标悬停BD Icon | 显示详细Tips |
| 鼠标移出Icon | Tips延迟0.15秒关闭 |
| 鼠标移动到Tips上 | Tips保持显示 |
| 点击Icon | 固定Tips，再次点击取消 |
| 右键Icon | 可打开该BD的详细成长树，后续版本可做 |
| 移动端长按Icon | 显示Tips |

---

### 6.2 Tips显示位置

Tips默认显示在BD Icon上方。

规则：

```text
优先显示在Icon上方
如果上方空间不足 → 显示在左上
如果左上超出屏幕 → 显示在右上
不能遮挡士兵抽屉核心按钮
不能遮挡AI当前音效进度条
```

## 8. Tips文案规则

### 8.1 基础属性BD Tips示例

#### 伤害提升

```text
名称：伤害强化
类型：基础属性
等级：Lv.4

当前效果：
AI角色所有伤害提高20%。

影响范围：
普通子弹、额外弹体、路径区域、AOE音波、状态伤害全部生效。

音效表现：
所有攻击音效音量略微增强，命中音更厚重。

特效表现：
弹体命中时产生更明显的冲击闪光。

下一级效果：
所有伤害提高至25%。

玩家反制建议：
减少低血单位，使用高血量或抗性单位承伤。
```

#### 节奏速度

```text
名称：节奏加速
类型：基础属性 / 节奏
等级：Lv.3

当前效果：
AI所有伤害释放节奏间隔缩短15%。

影响范围：
普通子弹、额外弹体、音波AOE、路径生成频率。

音效表现：
节拍点间隔变短，音效播放器推进更快。

特效表现：
攻击前摇更短，弹幕出现频率更高。

下一级效果：
节奏间隔缩短至20%。

玩家反制建议：
不要在AI连续攻击阶段召唤脆皮小怪，优先等待后摇窗口。
```

#### 发射数量

```text
名称：多重发射
类型：基础属性 / 弹幕数量
等级：Lv.2

当前效果：
每次释放伤害时，额外发射2个基础弹体。

影响范围：
普通子弹和部分可复制弹体。

音效表现：
同一个节拍点会出现连续短音，不是完全重叠播放。

特效表现：
弹体以小角度散射，形成轻微扇形。

下一级效果：
额外发射数量 +1。

玩家反制建议：
避免正面密集召唤，优先从侧翼召唤单位。
```

## 12. UI状态变化规则

### 12.1 AI获得新BD

流程：

```text
AI升级
↓
AI自动选择BD
↓
屏幕右侧或中下方弹出“获得BD”
↓
BD Icon飞入当前BD列表
↓
BD Icon闪光1秒
↓
如果该BD影响音轨，音效进度条短暂显示对应音轨变化
```

表现：

```text
新BD：火焰弹
Icon从升级弹窗飞入BD列表
火焰弹Icon带橙色闪光
Tips可立即悬停查看
```

---

### 12.2 AI升级已有BD

流程：

```text
AI选择已有BD升级
↓
列表中对应Icon播放升级闪光
↓
等级数字 +1
↓
右上角出现上箭头
↓
Tips中当前数值更新
```

---

### 12.3 BD被玩家克制 / 削弱

如果玩家释放造物主技能，削弱AI某类BD，需要在Icon上显示状态。

| 被削弱状态 | UI表现 |
|---|---|
| 火焰伤害降低 | 火焰BD Icon变暗 |
| 弹体速度降低 | 弹体类BD出现减速箭头 |
| 节奏变慢 | 节奏类BD出现断拍图标 |
| AOE范围降低 | AOE Icon外圈缩小 |
| 毒伤免疫 | 毒类BD出现裂纹遮罩 |

## 13. 当前BD列表与音效进度条联动

### 13.1 影响当前攻击的BD高亮

当AI当前音效进度条正在释放某种攻击时，相关BD Icon高亮。

例如：

AI正在释放火焰弹：

```text
音效进度条：中低音火焰节拍
BD列表：[火焰弹] Icon发光
```

AI正在释放闪电弹射：

```text
音效进度条：高频电流节拍
BD列表：[弹射闪电弹] Icon发光
```

---

### 13.2 节拍点触发时的Icon反馈

每次BD参与伤害触发，对应Icon轻微跳动。

| 事件 | Icon反馈 |
|---|---|
| 闪电弹发射 | 闪电BD Icon闪一下 |
| 火焰弹爆炸 | 火焰BD Icon外圈扩散 |
| 冰冻弹命中 | 冰冻BD Icon结霜 |
| 毒气区域生成 | 毒气BD Icon冒泡 |
| 音波AOE释放 | AOE BD Icon震动 |
| 节奏加速生效 | 节奏BD Icon快速脉冲 |

```text
显示初始BD：
```

```text

↓

↓
AIBDListPanel接收 OnAIBDAdded 事件

判断是否已有同ID BD
```

## 17. 玩家读取逻辑

玩家看到BD列表后，应该能快速判断AI当前流派。

### 17.1 示例：雷电流

显示：

```text
[节奏加速 Lv.3] [弹射闪电弹 Lv.4] [多重发射 Lv.2] [高频连奏 Lv.1]
```

玩家理解：

```text
AI攻击频率高
闪电会弹射
不能密集召唤小怪
需要分散召唤或用高血单位承伤
```

---

### 17.2 示例：火焰范围流

显示：

```text
[伤害强化 Lv.2] [火焰弹 Lv.4] [燃烧轨迹 Lv.3] [音波扩散 Lv.2]
```

玩家理解：

```text
AI范围清怪强
近身小怪容易被烧死
需要远程怪、火抗怪、分散召唤
```

---

### 17.3 示例：控制冰冻流

显示：

```text
[冰冻弹 Lv.3] [冰霜轨迹 Lv.2] [节奏减速重击 Lv.1] [AOE冰环 Lv.2]
```

玩家理解：

```text
AI控制能力强
精英怪容易被冻住
需要多单位分摊控制，或使用免控怪
```

## 19. 美术表现规则

### 19.1 Icon风格

建议：

- 小尺寸可读性强。
- 图形不要太复杂。
- 元素颜色明确。
- 每个类型的角标统一。
- 品质边框明显但不要太厚。
- 核心BD要比普通BD更突出。

---

| 元素 | Icon关键词 |
|---|---|
| 雷电 | 闪电、电弧、蓝白 |

## 20. 最终效果目标

这个UI的核心不是简单显示图标，而是让玩家形成判断：

```text
AI当前拿了什么BD
↓
AI当前弹幕会变成什么样
↓
AI当前音效节奏为什么变快 / 变密 / 变重
↓
我现在应该召唤什么怪
↓
我应该什么时候释放造物主技能
```

最终玩家应该能通过：

```text
音效进度条 + 当前BD图标 + Tips说明
```

快速判断AI的当前成长流派，并做出反制策略。
